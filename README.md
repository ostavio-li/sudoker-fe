#### 安装教程

---

##### 二进制安装

（项目启动阶段，尚无二进制产物）

##### 源码安装

1. 拉取代码

   ```shell
   git clone https://gitee.com/carlos_li/new-sudoker-fe.git
   ```

2. 安装依赖

   ```shell
   npm install
   ```

3. 开发模式启动

   ```shell
   npm run serve
   ```

4. 生产环境打包

   ```shell
   npm run build
   ```
